Styling & Shadow DOM

Shadow DOM beeinflusst Styling entscheidend

  • DOM ist isoliert
  • Styling ist auch isoliert

Scoped CSS

    Generell:
  • CSS Selektoren von ausserhalb nicht angewendet in Komponente
  • CSS Selektoren von innerhalb der Komponente nicht angewendet ausserhalb

:host

Mit dem :host Selektor kann sich ein Element selber stylen

  • Styles auf dem Element von aussen haben höhere Priorität
  • :host(selector), falls Element selector erfüllt

:host-context(selector)

Kommt zur Anwendung, falls Element
oder einen Ancestor selector erfüllt

:slotted(selector)

Kommt zur Anwendung, falls Element in Slot einen selector erfüllt
