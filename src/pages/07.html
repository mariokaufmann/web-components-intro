Framework Integration

Framework Integration nicht überall gleich gut

https://custom-elements-everywhere.com/

Angular

  • Gute Integration
  • Komplexe Properties können mit der gewohnten Syntax gesetzt werden
  • Kein Autocomplete, keine FormControls, keine generischen Komponenten
            
                  <vis-vote-counter
                    [counterTitle]="Likes given"
                    showVotes
                    [voterData]="voterData"
                    (countChanged)="handleCount($event)"
                  ></vis-vote-counter>

React

  • Daten werden immer als HTML Attribute gesetzt
  • Komplexe Properties können nur stringified übergeben werden
  • Events von Custom Elements können nur mit Workarounds abgehört werden
